# CLAIMS — factual ledger

**This file is background, not for publication.** If the post leans on
factual claims (statistics, historical anecdotes, citations), record
each one here with its verification status. The publishing agent uses
this to know which claims are settled and which need weakening.

Optional — skip this file if the post is opinion / essay only and
doesn't rest on a body of cited evidence.

## Verdicts

- ✅ **supported** — claim survived verification against primary sources.
- ⚠️ **supported with correction** — core claim survives but the popular
  framing is inflated. The publishing agent should use the sober version
  and (in the paper variant) foreground the correction.
- ❌ **refuted / myth** — do not include this claim in any artifact.
